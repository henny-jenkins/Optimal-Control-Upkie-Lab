This is Henry Jenkins' GitHub repo for the Upkie lab code, as part of the Optimal Control and Reinforcement Learning class at Harvard University.

The code to train the custom PPO policy is located at `Lab/upkie/train_custom_policy.py`. The code to roll out the custom PPO policy is located at `Lab/upkie/rollout_custom_policy.py`. The checkpoint models and final models are located at `Lab/upkie/models/checkpoints/` and `Lab/upkie/models/ppo_upkie_final.zip`, respectively.
